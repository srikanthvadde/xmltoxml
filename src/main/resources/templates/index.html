XML2XML TOOL:


Process :

Steps:

  • 1.Upload Required Excel file From the Location
  • 2.Click on the column button(It shows all the Excel Headers in Below Grid)
  • 3.Upload Required XML File from the Location
  • 4.Click on the column button(It shows all the XML Tags and Attributes in Below Grid)
  • 5.Map the Excel header with XML (Tag or attribute)(It allows only one at a time)
  • 6.The selected mapping will display in Mapping Grid
  • 7.Reverify the Mapping data before Generate
  • 8.Finally Click on Generate button
  • 9.Your Required XML is ready (It downloads into browser window).
  • 10.Happy Using!!!

Note: We have Delete Icon (Red colour) to delete the Accidental mappings from Mapping grid.
